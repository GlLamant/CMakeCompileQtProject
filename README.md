# CMakeCompileQtProject

---



## Dependency
- Qt-5.9.6
- cmake-2.8.11
  

## Build
```shell
mkdir build
cd build 
cmake ..
make
```

## Feature
- CMake complies Qt project.
- Remove the **console** of GUI program on Windows.
